# 🚀 Daily Activity Tracker - Premium Flask Module

A high-precision, UI/UX-focused activity logger built with **Python Flask** and **SQLite**. Designed with a "SaaS-style" white premium aesthetic, featuring real-time activity streaming, dynamic categorization, and a smooth checklist workflow.

## 🛠️ Tech Stack
- **Backend:** Python, Flask, Flask-SQLAlchemy
- **Database:** SQLite
- **Frontend:** HTML5, CSS3 (Modern UI/UX), JavaScript (ES6)
- **Icons:** Font Awesome (FAS/FAB)

## ✨ Features
- **Modular Architecture:** Separate files for Models, Controllers, and Assets.
- **Dynamic Module:** Real-time upload and state toggling without page clutter.
- **Premium UI:** Clean white aesthetic with soft shadows and interactive hover states.
- **Categorization:** Built-in dropdown for Work, Freelance, and Personal tracking.

## 📂 Project Structure
```text
daily_activity/
├── static/              # CSS and JS Assets
├── templates/           # HTML Jinja2 Templates
├── app.py               # Main Application Logic
├── models.py            # Database Schema
└── daily_activity.db    # SQLite Database (Local)

```

## 🚀 How to Run

1. **Activate Environment:** `.\venv\Scripts\activate`
2. **Install Deps:** `pip install flask flask-sqlalchemy`
3. **Run App:** `py app.py`
